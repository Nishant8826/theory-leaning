# Building Workflows with n8n

---

### What
**n8n** is an open-source workflow automation tool (similar to Zapier or Make). It provides a visual node-based editor that allows you to connect different APIs, databases, and AI models together by drawing lines between them, without writing heavy backend code.

---

### Why
Writing custom Node.js integrations for Gmail, Slack, Salesforce, and OpenAI is tedious and hard to maintain. n8n simplifies this visually. You can visually build complex AI Agent workflows—where receiving an email triggers an LLM, the LLM decides instructions, and n8n automatically executes those instructions via another API. 

---

### How
An n8n workflow consists of **Nodes**.
1. **Trigger Node:** Starts the workflow (e.g., A Webhook, a Schedule, or "On New Email").
2. **Action Nodes:** Performs a task (e.g., Send data to OpenAI, Fetch data from Postgres).
3. **Connections:** Lines between nodes that pass data (JSON) from one step to the next.

---

### Implementation

Instead of code, n8n is visual. But to connect a Next.js frontend to an n8n workflow, you use a **Webhook**. A Webhook is simply a URL generated by n8n that your app can send POST requests to.

```typescript
// Sending data from your React/Next.js app to trigger an n8n AI Workflow

async function triggerN8NChatbot() {
    // This URL is provided by the Webhook trigger node in n8n
    const n8nWebhookUrl = "https://your-n8n-instance.com/webhook/support-bot";

    const payload = {
        userId: "123",
        message: "Hi, I need help resetting my password.",
        channel: "web"
    };

    console.log("Triggering n8n visual AI workflow...");
    
    // Send the data to n8n
    const response = await fetch(n8nWebhookUrl, {
        method: "POST",
        headers: { "Content-Type": "application/json" },
        body: JSON.stringify(payload)
    });

    const data = await response.json();
    console.log("n8n processed the workflow and returned:", data.reply);
}
```

---

### Steps (Example: AI Chatbot Automation in n8n)
1. **Start:** Add a "Webhook" trigger node in n8n.
2. **AI Action:** Add an "OpenAI" node. Connect the Webhook to it. Configure it to read the `message` from the Webhook payload.
3. **Logic:** Add an "If" node. If the AI determines the sentiment is "Angry", route it to a "Slack" node.
4. **Action:** The "Slack" node sends a message to the Admin channel: "Angry client!".
5. **Response:** Route the flow back to the "Webhook Response" node, returning the AI's answer to your Next.js app.

---

### Integration

* **React/Next.js:** Do not write thousands of lines of API integration code. Instead, write a simple `fetch()` POST request pointing to your n8n webhook. 
* **Node.js backend:** Your backend might not be needed at all! For small tasks, React can talk directly to n8n, and n8n handles all the database and AI API calls instantly.

---

### Impact
n8n radically speeds up development. You can prototype a complex AI Agent that reads emails, generates summaries, updates Trello boards, and texts you on WhatsApp in about 30 minutes, all visually.

---

### Interview Questions
1. **What is an n8n Webhook?**
   *Answer: An HTTP endpoint automatically created by n8n that allows external applications to trigger an n8n workflow by sending a POST request to it.*
2. **Why use n8n instead of writing Node.js integrations manually?**
   *Answer: It abstracts away API authentication, significantly reduces boilerplate code, and provides a clear visual map of complex system architectures.*
3. **Name the core components of an n8n workflow.**
   *Answer: Triggers (events that start the flow) and Nodes (actions that process data or connect to services).*

---

### Summary
* n8n replaces massive backend boilerplate with a visual node editor.
* Workflows are typically triggered via Webhooks.
* Perfect for rapidly building complex AI interactions across multiple SaaS platforms (Slack, Gmail, HubSpot) without coding the APIs manually.
