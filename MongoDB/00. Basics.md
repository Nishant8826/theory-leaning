# MongoDB
### 1. No-SQL Document Based Database
### 2. Strong Support for Aggregation Pipes
### 3. Works on BSON format
### 4. Make our data persist

## Basic mongo commands on mongo shell
- To connect with mongoDB.
```sh
mongosh "mongodb+srv://<userName>:<password>@<cluster>"
```
- To show all database inside cluster.
```sh
show dbs
```
- To use DB.
```sh
use <db_name>
```
- To show all collections in a DB.
```sh
show collections
```
- To fetch all documents.
```sh
db.collection_name.find({})
```
- To fetch a document.
```sh
db.collection_name.findByOne({key_name:"value_name"})
```
