`Desirable properties of Distributed Systems with replicated Data.`

# `C`onsistency 
Data should be same in all nodes.
# `A`vailability 
System should be available
# `P`artition Tolerance 
System can keep operating in case of network/sub-systems failures.
##
### Only 2 can be achieved for a system.
### Must have `P`artition Tolerance in a system because system should always be running and up.
Therefor system can have only CP or AP.
