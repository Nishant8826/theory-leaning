## 🔑 Quick Reference: Most Used Hooks

| Hook | Use For |
|---|---|
| `useState` | Local component state |
| `useEffect` | Side effects, API calls, timers |
| `useRef` | DOM access, persisting values |
| `useContext` | Reading from Context |
| `useMemo` | Cache expensive computations |
| `useCallback` | Cache function references |
| `useReducer` | Complex state with actions |
| `useNavigate` | Programmatic routing (React Router) |
| `useParams` | URL parameters (React Router) |
